Nosotros

Quienes somos Dev Works?

Estamos ubicados en Santa Cruz y nos dedicamos a ofrecer soluciones de desarrollo de software adaptadas a las necesidades de nuestros clientes. Nuestro enfoque está en crear herramientas eficientes y personalizadas que impulsen el crecimiento y la digitalización de los negocios.

Tecnologias

Tecnologias de nuestro Desarrollo

Flutter

Framework de Google

Flutter permite crear aplicaciones móviles nativas con un solo código base.

React

Librería de JavaScript

React es una biblioteca popular para construir interfaces de usuario dinámicas y eficientes.

PHP

Lenguaje de Programación

PHP es uno de los lenguajes más utilizados para el desarrollo web backend.

Laravel

Framework de PHP

Laravel es un potente framework PHP que simplifica el desarrollo web con herramientas avanzadas.

WordPress

Sistema de Gestión De Contenidos

(CMS, por sus siglas en inglés) que permite crear y administrar sitios web de manera sencilla, sin necesidad de conocimientos avanzados de programación.

SQL Server

Sistema de Bases de Datos

SQL Server es un robusto sistema de bases de datos de Microsoft, ideal para grandes organizaciones.

NestJS

Framework de JavaScript

NestJS es un framework de desarrollo backend para Node.js que utiliza TypeScript y está inspirado en la arquitectura de Angular.

Docker

Contenedor de Proyectos

Docker es una plataforma que permite empaquetar aplicaciones y sus dependencias en contenedores.
